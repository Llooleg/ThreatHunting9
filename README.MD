# Исследование структуры и глобальной связности Интернета (маршрутизация)

Проект посвящён исследованию актуальной структуры Интернета на уровне автономных систем (AS) и анализу маршрутов прохождения сетевого трафика между ними.

Целью является разработка программного пакета на языке **R** для сбора, обработки, анализа и визуализации данных об автономных системах, их связности и географическом распределении на основе открытых источников и результатов сетевых измерений.

---

## Идея проекта

На основе открытых источников информации:

- **DB-IP / MaxMind GeoLite**  
  - https://db-ip.com/  
  - https://www.maxmind.com/

а также данных о маршрутах прохождения сетевого трафика между автономными системами (AS), полученных с помощью системных инструментов (например, `traceroute` и аналогов),

проводится исследование:
- структуры Интернета,
- глобальной связности автономных систем,
- маршрутизации и взаимного расположения AS,
- географического распределения сетевых подсетей.

---

## Основные задачи

Проект включает следующие этапы:

1. **Разработка средств сбора данных**
   - скачивание открытых наборов данных,
   - импорт и первичная очистка,
   - приведение данных к согласованному формату.  
   Данные сохраняются в директории `/data-raw`.

2. **Извлечение маршрутов передачи трафика**
   - парсинг результатов работы системных инструментов (`traceroute` и аналогов),
   - восстановление цепочек автономных систем на маршрутах.

3. **Формирование единого представления данных**
   - объединение информации об автономных системах,
   - указание IP-подсетей,
   - добавление географического положения,
   - сохранение результатов в виде единого `data.frame` с возможностью экспорта в CSV.

4. **Визуализация данных**
   - интерактивные графы связности AS,
   - карты маршрутизации и географического распределения,
   - веб-интерфейс на базе **Shiny** или **Quarto**.

5. **Контейнеризация**
   - подготовка `Dockerfile` для воспроизводимого окружения.

6. **Запуск веб-приложения**
   - конфигурация и запуск с использованием  
     `docker compose` или `podman`.

---

## Технологии и инструменты

- Язык программирования: **R**
- Работа с данными: `data.frame`, tidyverse
- Сетевые измерения: `traceroute` и аналоги
- Геоданные: DB-IP, MaxMind GeoLite
- Визуализация: Shiny / Quarto
- Контейнеризация: Docker / Podman
